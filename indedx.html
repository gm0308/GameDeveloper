We are 2D game Developer!


우리는 최고의 2D 게임을 만들기 위해 노력합니다

우리의 게임은 사용자들에게 최고의 경험을 선사살 할 것입니다.



게임기획

게임기획은 참으로 고되고 힘든 일입니다. 이곳에는 아무말이나 채워넣아야만합니다. 그래야만 그럴듯해 보입니다. 더미 텍스트를 넣어봅시다.

게임 맵 설계하기

게임 맵 설계는 게임의 세계관을 설계하는 것입니다. 이곳에는 아무말이나 채워넣아야만합니다. 그래야만 그럴듯해 보입니다. 더미 텍스트를 넣어봅시다.

레벨디자인

레벨의 디자인은 플레이어의 경험을 만드는 작업니다. 이곳에는 아무말이나 채워넣아야만합니다. 그래야만 그럴듯해 보입니다. 더미 텍스트를 넣어봅시다.

2D Game Dev Engine


완전히 새로워진 2D 게임엔진을 만나보세요. 우리의 2D엔진은 기존의 엔진과 다르게 물리 계산에 필요한 시간을 최소로 단축하였습니다. 이를 통해 파워풀함 여러분만의 게임을 작설할 수 있습니다.
